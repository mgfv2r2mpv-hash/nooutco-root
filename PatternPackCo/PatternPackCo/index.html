Pattern Pack Co. — finish packing the box

A gift box rolls down the line already part-packed with a repeating pattern (e.g. ❤️ ⭐ ❤️ ⭐ …). The learner fills the empty slots from the bulk bin; a finished box gets a bow and ships out.

  • Product Line — pick an emoji category (animals, food, shapes, etc.).
  • Pattern Length — items in the repeating unit (2 = AB, 3 = ABC).
  • Shown Reps — full repeats pre-packed before the empty slots.
  • Blanks to Fill — how many slots the learner packs.
  • Bank Size — total choices in the bulk bin.
  • ⚙ Settings — errorless, re-present errors, prompts, and motion options.

Set the options, then tap Start.
