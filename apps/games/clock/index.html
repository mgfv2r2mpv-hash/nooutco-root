Hickory Dickory Dock — receptive find-by-name

Pictures circle a cuckoo clock. As the rhyme plays, the target word appears — the learner taps the matching picture before the clock strikes the hour. Same skill as Receptive Words, with a nursery-rhyme wrapper for engagement.

  • Topic — pick an image category.
  • Array Size — how many pictures circle the clock (1–10).
  • Targets — open the Targets panel to pick which words can be the correct answer.
  • ⚙ Settings — turn animations off for fast-paced trials, plus errorless, re-present errors, and prompts.

Set Topic, Array Size, and Targets, then tap Start.
