Sequences & Patterns — continue the pattern

A repeating pattern (e.g. ❤️ ⭐ ❤️ ⭐ …) appears in boxes; the learner picks the next item(s) from the bank.

  • Symbol Set — pick an emoji category (animals, food, shapes, etc.).
  • Pattern Length — items in the repeating unit (2 = AB, 3 = ABC…).
  • Shown Reps — full repeats shown before the blanks.
  • Blanks to Fill — how many items the learner supplies.
  • Bank Size — total choices in the bank.
  • ⚙ Settings — errorless, re-present errors, prompts, and delay options.

Set the options, then tap Start.
