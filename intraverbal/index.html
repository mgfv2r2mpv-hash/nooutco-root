Intraverbal — fill-in-the-blank

The learner is given a carrier phrase (e.g., "You sleep in a ___") and selects the word that completes it from the array.

  • Category — pick the intraverbal category (animals, daily routines, etc.).
  • Array Size — number of choices shown (2–10).
  • Targets — open the Targets panel to pick which answers can appear as correct.
  • 🎙️ Record — record your own audio for each carrier phrase and target answer (optional).
  • ⚙ Settings — errorless, re-present errors, prompts.

Set Category, Array Size, and Targets, then tap Start.
