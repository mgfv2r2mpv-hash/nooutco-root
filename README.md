# No Outcome

**Progress is a process.**

No Outcome is a passion project at [nooutco.me](https://nooutco.me),
built by NoOutcomes ABA to bridge Applied Behavior Analysis and
technology. It is a set of small, focused tools for the people who
actually do the work: clinicians, supervisors, learners, and the
families around them.

## Why this exists

ABA providers spend their off-hours cutting, laminating, and re-printing
the same targets for the same learners, over and over, because the
tools to do the science _right_ haven't existed in a form that fits a
session. Assessments that should be fast are slow. Data that should be
obvious gets lost. BCBAs are already stretched thin, and the families
they serve are already carrying real costs for care in a space that has
stayed low-tech for too long.

This project exists to close that gap.

The aim is simple:

- **Bring researched ABA into a format that is actually fun to use** -
  for clients in session, and for the clinicians running it.
- **Capture the useful data around treatment that traditional paper
  and pencil methods quietly lose** - the context, the latencies, the
  prompts, the patterns - without making anyone do extra work.
- **Tackle the problems whose computational or implementational
  difficulty has kept them out of the clinic**, so the science can show
  up where it's needed.
- **Give back the evenings** that get eaten by prep work, so the people
  delivering treatment can rest.

This is built from years of formal training in math, ABA, and computer
science, and from years of working on both sides - in tech and in ABA -
with the conviction that the two should not be strangers to each other.

## What is here

- [`games.nooutco.me`](https://games.nooutco.me) - fun, session-ready
  activities for clinicians and clients.
- [`tools.nooutco.me`](https://tools.nooutco.me) - supervision and
  assessment utilities for BCBAs and supervisors.

More is coming. This is an active, evolving passion project.
