Feature · Function · Class

The learner hears a feature, function, or category prompt (e.g., "something you eat with") and picks the matching item from the array.

  • Mode — Feature, Function, Class (within group), or Class (cross-category).
  • Array Size — number of choices shown (2–10).
  • Tag — vocabulary set to draw items from.
  • Targets — open the Targets panel to pick which items can be the correct answer.
  • ⚙ Settings — errorless, re-present errors, prompts.

Set Mode, Array Size, Tag, and Targets, then tap Start.
